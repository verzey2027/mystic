# MysticG — Examples Pack (TH) — งาน/ความรัก/Matching (v1)

> ชุดตัวอย่างสำหรับเทสต์บอท + ทำ few-shot prompting

## 1) Work — Tarot (3-card)
**Q:** ควรย้ายงานใน 3 เดือนนี้ไหม
**A (ละมุนแต่ฟันธง):**
- **สรุป:** ย้ายได้ แต่ต้องย้ายแบบวางแผน—อย่าให้ความเหนื่อยพาเธอตัดสินใจเร็วเกิน
- **โอกาส:**
  - ได้ทีมใหม่ที่โตไว และได้โชว์ฝีมือจริง
  - มีเกณฑ์ได้โปรเจคที่ดันโปรไฟล์
- **ระวัง:**
  - งานที่พูดสวย แต่ขอบเขตงานไม่ชัด
  - สัญญา/เงื่อนไขต้องอ่านให้ครบ
- **ทำเลย:**
  - อัปเดตพอร์ต 3 ชิ้นที่ขายได้ที่สุด
  - ลิสต์ must-have 5 ข้อก่อนรับ offer
  - นัดคุยเงินเดือน/บทบาทให้ชัดตั้งแต่รอบแรก
- **เวลา:** 1–6 สัปดาห์จะเห็นสัญญาณเรียกคุยชัด

## 2) Work — Feng Shui (โต๊ะทำงาน)
**Q:** ทำงานที่บ้านแล้วไม่ค่อยโฟกัส งานไม่เดิน
**A:**
- **สรุป:** งานไม่เดินเพราะพลังงานรอบตัว “รก-ล้า” มากกว่าเธอไม่เก่ง—แค่จัดจุดหลัก งานจะลื่นขึ้นทันที
- **โอกาส:**
  - พอจัดโต๊ะ/แสง เธอจะกลับมาทำงานได้ยาวและนิ่ง
- **ระวัง:**
  - วางโต๊ะหันชนกำแพง/หลังโล่ง ทำให้ใจล้าแบบไม่รู้ตัว
- **ทำเลย:**
  - เคลียร์หน้าโต๊ะ 70% ให้เหลือของจำเป็น
  - เพิ่มไฟสว่างที่หน้าโต๊ะ (ไม่ย้อนแสง)
  - ทำ “หลังพิง” เช่น เก้าอี้พนักสูง/ชั้นเตี้ยด้านหลัง
- **เวลา:** ภายใน 3 วันจะเริ่มเห็นผลเรื่องสมาธิ

## 3) Love — Tarot (relationship)
**Q:** คนคุยยังไม่ชัดเจน เราควรไปต่อไหม
**A:**
- **สรุป:** ไปต่อได้ ถ้าเธอคุยเรื่อง “สถานะและขอบเขต” ให้ชัด—ไม่งั้นความสัมพันธ์จะวนลูปเดิมจนเธอเหนื่อย
- **โอกาส:**
  - อีกฝ่ายมีใจ แต่ยังลังเลเรื่องความรับผิดชอบ
- **ระวัง:**
  - การเดาใจจะทำให้เธอเสียพลัง
- **ทำเลย:**
  - คุยให้ชัดว่าอยากได้ความสัมพันธ์แบบไหน
  - ตั้งเดดไลน์ใจตัวเอง 2–4 สัปดาห์
- **เวลา:** 2–4 สัปดาห์จะรู้คำตอบชัด

## 4) Matching — Chinese Zodiac (light)
**Q:** เข้ากันได้ไหม (ขอปีเกิดทั้งคู่)
**A:**
- **สรุป:** เข้ากันได้ แต่ต้องแบ่งบทบาทให้ชัด—คนหนึ่งนำ คนหนึ่งซัพพอร์ต แล้วจะพาไปไกล
- **จุดแข็ง:** สื่อสารไว / ช่วยกันแก้ปัญหา
- **จุดเสี่ยง:** ต่างคนต่างดื้อในช่วงเครียด
- **ไปต่อยังไง:** นัดคุยกติกาเวลาทะเลาะ + ตั้งเป้าหมายร่วม 1 เรื่อง
