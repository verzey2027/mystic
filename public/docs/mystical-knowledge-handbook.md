## 1. ภาพรวมระบบ

### 1.1 ศาสตร์ที่รองรับ

| ศาสตร์ | รายละเอียด | ข้อมูลที่ใช้ |
|--------|------------|--------------|
| **ไพ่ทาโรต์** | 78 ใบ (Major 22 + Minor 56) | ความหมายตรง/กลับ ตามชุด |
| **ไพ่จิตวิญญาณ** | ไพ่ประจำวันเกิด + ราศี | Life Path Number |
| **เลขศาสตร์** | เบอร์โทรศัพท์ | คะแนน 0-99, Root Number |
| **ดวงชะตา** | รายวัน/สัปดาห์/เดือน | ตำแหน่งดาว |
| **ปีจีน** | นักษัตร 12 ปี | ธาตุ 5 ธาตุ |
| **ความเข้ากัน** | ความรัก/มิตรภาพ/การงาน | องค์ประกอบที่เข้ากัน |

### 1.2 โครงสร้าง AI Reading

```
Input → Processing → AI Interpretation → Output
```

- **Input**: คำถาม/วันเกิด/เบอร์โทร/ไพ่ที่สุ่มได้
- **Processing**: ระบบเลือก template และ context
- **AI**: สร้างคำทำนายเป็นธรรมชาติ
- **Output**: ข้อความ 3-5 ย่อหน้า + คำแนะนำ

## 2. ไพ่ทาโรต์ (Tarot)

### 2.1 โครงสร้างสำรับ

**Major Arcana (22 ใบ)**
- ไพ่หลักที่มีความหมายลึกซึ้ง
- แทนชีวิตจิตวิญญาณและบทเรียนสำคัญ

**Minor Arcana (56 ใบ)**
- แบ่งเป็น 4 ชุด (Suits): ถ้วย, ดาบ, ไม้เท้า, เหรียญ
- แต่ละชุดมี 14 ใบ (Ace ถึง King)

| ไพ่ | ชื่อ | ความหมายตรง | ความหมายกลับ |
|-----|------|-------------|--------------|

### 2.3 ชุดไพ่ Minor Arcana (4 Suits)

**Cups (ถ้วย)** - อารมณ์ ความรัก ความสัมพันธ์
- Ace: ความรักใหม่
- 2-10: การพัฒนาความสัมพันธ์
- Page: ข้อความแห่งความรัก
- Knight: การเสนอความรัก
- Queen: ความรักที่อิ่มเอม
- King: การควบคุมอารมณ์

**Swords (ดาบ)** - ความคิด การตัดสินใจ ความขัดแย้ง
- Ace: ความคิดใหม่
- 2-10: การต่อสู้ทางใจ
- Page: ข่าวสาร
- Knight: การตัดสินใจเร็ว
- Queen: ความยุติธรรม
- King: อำนาจทางความคิด

**Wands (ไม้เท้า)** - พลังงาน การสร้างสรรค์ การลงมือทำ
- Ace: แรงบันดาลใจ
- 2-10: การพัฒนาโครงการ
- Page: ข้อเสนอใหม่
- Knight: การกระทำ
- Queen: ความมั่นใจ
- King: ความสำเร็จ

**Pentacles (เหรียญ)** - เงิน การงาน ความมั่นคง
- Ace: โอกาสทางการเงิน
- 2-10: การจัดการเงิน
- Page: การศึกษาใหม่
- Knight: ความมุ่งมั่น
- Queen: ความอุดมสมบูรณ์
- King: ความมั่งคั่ง

### 2.4 สเปรดไพ่ (Spreads)

**ไพ่ 1 ใบ**: คำตอบรวดเร็ว
**ไพ่ 3 ใบ**: อดีต - ปัจจุบัน - อนาคต
**ไพ่ 5 ใบ**: ตำแหน่งแบบกาง
**ไพ่ 10 ใบ (Celtic Cross)**: วิเคราะห์ลึก

```
Life Path Number = ผลรวมของตัวเลขวันเกิด

|--------|-------------|-----------|

| 2 | The High Priestess | สัญชาตญาณ, ความอ่อนโยน |
| 3 | The Empress | ความคิดสร้างสรรค์, การสื่อสาร |
| 4 | The Emperor | ความมั่นคง, ความเป็นระเบียบ |
| 5 | The Hierophant | อิสระ, การผจญภัย |
| 6 | The Lovers | ความรัก, ความรับผิดชอบ |
| 7 | The Chariot | การค้นหา, ปัญญา |

## 4. เลขศาสตร์ (Numerology)

### 4.1 การคำนวณเบอร์โทร

```
คะแนน = ผลรวมพลังตัวเลขแต่ละตัว
Root Number = ผลรวมจนเหลือตัวเดียว
```

### 4.2 ตารางคะแนน

| คะแนน | ระดับ | ความหมาย |
|-------|-------|----------|
| 90-99 | S | ดีเยี่ยม |
| 80-89 | A | ดีมาก |
| 70-79 | B+ | ดี |
| 60-69 | B | ปานกลาง |
| 50-59 | C+ | พอใช้ |
| 40-49 | C | ควรระวัง |
| 0-39 | D | ควรเปลี่ยน |

### 4.3 ความหมายเลข 0-9

| เลข | พลัง | ด้านดี | ด้านระวัง |
|-----|------|--------|-----------|
| 1 | ผู้นำ | กล้าหาญ, อิสระ | ดื้อรั้น, เห็นแก่ตัว |
| 2 | ความสมดุล | ร่วมมือ, อ่อนโยน | ลังเล, พึ่งพา |
| 3 | สร้างสรรค์ | สนุกสนาน, สื่อสาร | กระจาย, ไม่จริงจัง |
| 4 | มั่นคง | ซื่อสัตย์, ระเบียบ | ดื้อรั้น, กลัวการเปลี่ยนแปลง |
| 5 | อิสระ | ยืดหยุ่น, กล้า | ไม่รับผิดชอบ, ขี้เกียจ |
| 6 | ความรัก | ดูแล, มีความรับผิดชอบ | วิตกกังวล, เห็นแก่ตัว |
| 7 | ปัญญา | วิเคราะห์, จิตวิญญาณ | ห่างเหิน, สงสัย |
| 8 | อำนาจ | ทุน, ความสำเร็จ | โลภ, กดขี่ |
| 9 | ความเมตตา | ให้, ความเข้าใจ | สูญเสีย, สิ้นหวัง |
| 0 | พลังจักรวาล | ศักยภาพไม่จำกัด | ความว่างเปล่า |

### 4.4 การวิเคราะห์ตำแหน่ง

- **ตำแหน่ง 1-3**: งาน/อาชีพ
- **ตำแหน่ง 4-6**: เงิน/การเงิน
- **ตำแหน่ง 7-9**: ความรัก/ความสัมพันธ์

## 8. โครงสร้างการตอบของ AI

### 8.1 Template Structure

```
1. บทนำ (เปิดใจ)
2. วิเคราะห์สถานการณ์
3. คำแนะนำ (3-5 ข้อ)
4. สรุป/กำลังใจ
```

### 8.2 หลักการเขียน

- ใช้ภาษาที่เข้าใจง่าย
- ไม่ตัดสินใจแทนผู้ใช้
- ให้กำลังใจเป็นหลัก
- ไม่ทำนายโชคชะตาแน่นอน
- แนะนำทางเลือก

### 8.3 ตัวอย่าง Prompt

```
คุณเป็นนักดูดวงที่มีประสบการณ์ กรุณาอ่านไพ่ให้ผู้ใช้

ไพ่ที่ได้: {cards}
คำถาม: {question}

กรุณาเขียนคำทำนายที่:
1. เข้าใจง่าย ไม่ใช้ศัพท์เทคนิคมาก
2. มีความยาว 3-5 ย่อหน้า

## 9. คำศัพท์และ Terminology

### 9.1 ศัพท์ไพ่ทาโรต์

- **Arcana**: ความลับ, ความรู้ลึกลับ
- **Upright**: ไพ่ตรง (ความหมายปกติ)
- **Reversed**: ไพ่กลับหัว (ความหมายตรงกันข้าม/ซ่อนเร้น)

- **Suit**: ชุดไพ่

### 9.2 ศัพท์เลขศาสตร์

- **Life Path Number**: เลขเส้นทางชีวิต
- **Root Number**: เลขราก
- **Master Number**: เลขมาสเตอร์ (11, 22)
- **Expression Number**: เลขการแสดงออก

### 9.3 ศัพท์ดวงชะตา

- **Ascendant**: ลัคนาราศี
- **Moon Sign**: ราศีจันทรคติ
- **Sun Sign**: ราศีสุริยคติ
- **House**: เรือน

### 9.4 ศัพท์ปีจีน

- **นักษัตร**: สัตว์ประจำปีเกิด
- **ธาตุ**: ธาตุประจำปี
- **ยิ่ง/อี่**: ปีที่หนึ่งของวัฏจักร
