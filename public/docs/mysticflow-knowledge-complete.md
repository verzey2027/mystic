## 1. ภาพรวมระบบ

### 1.1 วัตถุประสงค์
- รวมองค์ความรู้ด้านศาสตร์ทำนายที่ได้รับความนิยมในไทย เพื่อให้ AI สามารถตอบคำถามและให้คำแนะนำได้สอดคล้องกับวัฒนธรรม
- กำหนดโทน **"มูละมุน + ฟันธงอย่างมั่นใจ"** พร้อมข้อห้ามและแนวทางการสื่อสารเชิงบวก
- สร้างโครงสร้างข้อมูลมาตรฐานสำหรับการเก็บอินพุต การประมวลผล และรูปแบบข้อความผลลัพธ์

### 1.2 โครงสร้าง AI Reading
```
Input → Processing → AI Interpretation → Output
```

| ขั้นตอน | รายละเอียด |
|----------|-----------|
| **Input** | คำถาม/วันเกิด/เบอร์โทร/ไพ่ที่สุ่มได้ |
| **Processing** | ระบบเลือก template และ context |
| **AI** | สร้างคำทำนายเป็นธรรมชาติ |
| **Output** | ข้อความ structured + คำแนะนำ |

## 2. หลักการจริยธรรมและความปลอดภัย

### 2.1 หลักการหลัก

1. **การใช้เพื่อความเชื่อและการสะท้อนใจ** — ไม่ใช่วิทยาศาสตร์หรือหลักฐานทางการแพทย์/การเงิน
2. **ห้ามเนื้อหาที่สร้างความกลัวสุดโต่ง** — ข่มขู่ สาปแช่ง หรือรับประกันผลแน่นอน
3. **ไม่ให้คำแนะนำด้านการแพทย์ กฎหมาย** — หรือวิธีหลบเลี่ยงกฎหมาย
4. **ทุกคำทำนายต้องมี actionable advice** — อย่างน้อย 1–3 ข้อ
5. **ความเป็นส่วนตัว** — วัน/เวลา/สถานที่เกิด เลขติดต่อ รูปใบหน้า/ฝ่ามือ เป็นข้อมูลอ่อนไหว

### 2.2 Tone & Style
- **style**: มูละมุน
- **delivery**: ฟันธงตรงๆ
- **ต้องมี**: actionable_advice
- **หลีกเลี่ยง**: fearmongering, fatalism, guarantees

## 3. โครงสร้างคำตอบมาตรฐาน

### 3.1 Message Format (ทั่วไป)

| ส่วน | รายละเอียด |
|------|-----------|
| **Summary** | สรุปภาพรวม 2–4 บรรทัด |
| **Opportunities** | จุดเด่นหรือโอกาส 1–3 bullet |
| **Risks** | สิ่งที่ต้องระวัง 1–3 bullet |
| **Actions** | คำแนะนำลงมือทำ 1–3 bullet |
| **Timeframe** | กรอบเวลา (ถ้าศาสตร์ทำได้) |
| **Confidence/Disclaimer** | ระดับความมั่นใจ (low/med/high) |

### 3.2 Special Formats

**สำหรับ Tarot:**
- เพิ่มรายการไพ่ (position / card / keywords / meaning)

**สำหรับ Feng Shui:**
- Quick Fixes + Deep Changes + เหตุผลเชิงภาษาคน

**อินพุต:**
- ประเภทสถานที่ (บ้าน/ออฟฟิศ)
- ปัญหาที่พบ
- ผัง/ทิศทาง (ถ้ามี)

**เอาต์พุต:**
- Summary
- Quick Fixes (แก้ไขง่าย รวดเร็ว)
- Deep Changes (เปลี่ยนแปลงระยะยาว)
- เหตุผลเชิงภาษาคน

**หลักการ:**
- เริ่มจากการแก้ปัญหาแบบงบต่ำก่อน

### 5.8 ดูลายมือ (Palmistry)

**อินพุต:** รูปฝ่ามือหรือคำบรรยายเส้นสำคัญ

**เอาต์พุต:**
- จุดเด่น
- จุดที่ต้องระวัง
- คำแนะนำพักผ่อน

**ข้อห้าม:**
- ห้ามผูกกับการแพทย์หรือโรคร้าย

### 5.9 โหงวเฮ้ง (Face Reading)

**อินพุต:** รูปหน้า/คำบรรยาย (ไม่บังคับ)

**เอาต์พุต:** คำแนะนำการนำเสนอ/สื่อสาร

**ข้อควรระวัง:**
- หลีกเลี่ยงเหมารวม/เหยียด

### 5.10 ปฏิทินจันทรคติ + ฤกษ์ยาม

**อินพุต:**
- กิจกรรมที่ต้องการทำ
- ช่วงวันที่สะดวก
- ข้อจำกัด

**เอาต์พุต:**
- 2–3 ช่วงเวลาพร้อมเหตุผล
- แผนสำรอง

**เหตุผลภาษาคน:**
- วันธงชัย
- วันอธิบดี
- วันมหาสูตร

## 7. ขั้นตอนและ Best Practices

### 7.1 ขั้นตอนการทำงาน

1. **เก็บข้อมูลอินพุต**
   - ผ่านฟอร์มที่แยกตามศาสตร์
   - พร้อม validation
   - เวลาเกิดไม่จำเป็น → ปรับความมั่นใจ

2. **เลือก Template/Persona**
   - Work Reading
   - Love Reading
   - Matching
   - Career Matching

3. **Pre-processing**
   - สุ่มหรือจัดไพ่ตามกฎ
   - คำนวณ Life Path Number
   - คำนวณคะแนนเลขศาสตร์

4. **Inference**
   - ป้อนข้อมูล + prompt โครงสร้าง
   - LLM เติมเนื้อหาเชิงมนุษย์

5. **Post-processing & Safety Check**
   - ตรวจคำต้องห้าม
   - เติม disclaimer
   - ให้คำแนะนำ actionable ครบถ้วน

### 7.2 Maintenance

- ทบทวนความถูกต้องทุก 3 เดือน
- เก็บ feedback ผู้ใช้
- เพิ่มศาสตร์ใหม่ตามความนิยม
- อัปเดต tone/ตัวอย่างให้ทันสมัย

## 8. คำศัพท์และ Terminology

### 8.1 ศัพท์ไพ่ทาโรต์
- **Arcana**: ความลับ, ความรู้ลึกลับ
- **Upright**: ไพ่ตรง (ความหมายปกติ)
- **Reversed**: ไพ่กลับหัว (ความหมายตรงกันข้าม)
- **Spread**: การวางไพ่
- **Suit**: ชุดไพ่

### 8.2 ศัพท์เลขศาสตร์
- **Life Path Number**: เลขเส้นทางชีวิต
- **Root Number**: เลขราก
- **Master Number**: เลขมาสเตอร์ (11, 22)
- **Expression Number**: เลขการแสดงออก

### 8.3 ศัพท์ดวงชะตา
- **Ascendant**: ลัคนาราศี
- **Moon Sign**: ราศีจันทรคติ
- **Sun Sign**: ราศีสุริยคติ
- **House**: เรือน

### 8.4 ศัพท์ปีจีน
- **นักษัตร**: สัตว์ประจำปีเกิด
- **ธาตุ**: ธาตุประจำปี
- **ยิ่ง/อี่**: ปีที่หนึ่งของวัฏจักร
- **Wu Xing**: ธาตุ 5 ธาตุ (ไม้ ไฟ ดิน โลหะ น้ำ)
