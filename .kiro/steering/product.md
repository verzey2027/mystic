# Product Overview

REFFORTUNE is a Thai-first online fortune-telling web application that provides three core divination services:

1. Tarot Reading (1/3/10 card spreads)
2. Spirit Card (based on date of birth)
3. Numerology Analysis (Thai phone number analysis)

## Core Value Proposition

- Clear, actionable guidance (not vague prophecy)
- Fast results (<5 seconds interaction time)
- Thai language first with natural, conversational tone
- Deterministic baseline interpretations for consistency
- Trust transparency showing evidence and confidence

## User Journey

Awareness → Reading → Trust → Share → Paid → Retain

Key features:
- 78-card tarot library
- Follow-up chat mode for deeper questions
- Floating action button (FAB) with LINE add CTA on every page
- Credit-based monetization with paywall triggers
- Reading history and saved readings

## Target Audience

Thai users seeking practical life guidance for work, money, and relationships through divination systems.
